# 游戏音效文件

这个目录包含《外星人入侵》游戏的音效文件。如果没有这些文件，游戏仍然可以运行，但没有声音效果。

## 需要的音效文件：

1. **shoot.wav** - 玩家飞船射击声音
2. **alien_shoot.wav** - 外星人射击声音  
3. **explosion.wav** - 爆炸声音（飞船被击中或外星人被消灭）
4. **background.mp3** - 背景音乐

## 音效文件格式：

- 音效文件支持 WAV、MP3、OGG 格式
- 建议使用较小的文件大小以提高游戏性能
- 音量已在代码中调整，可以根据需要修改

## 代码中的音效控制：

- 玩家射击：`self.shoot_sound.play()`
- 外星人射击：`self.alien_shoot_sound.play()`
- 爆炸：`self.explosion_sound.play()`
- 背景音乐：循环播放，音量30%

如果没有音效文件，游戏会自动处理错误并继续运行。
